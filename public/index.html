Form Testserver

de

Dieser Server ist zum Testen vom Formularen gedacht. Die Formelemente werden auf der Console geloggt. hochgeladene Dateien werden in /uploads/ gespeichert. Alte Dateien werden überschrieben.

Zur Benutzung muss ein POST Requast auf "/" abgesetzt werden. Cors ist aktiviert. Es sollte auch von anderen Origins gepostet werden können.

<form action="http://localhost:4000/" method="POST" enctype="multipart/form-data">...</form>

So kann es aussehen.

Ein POST auf "/" gibt abwechselnd ein json mit status:"ok" und status:"fail" zurück. Ein POST auf "/ok" gibt immer ok, und "/fail" immer fail.

en

This server is made to test forms. The form data get logged to the console. The Files get saved in /uploads/ . New files with the same name override the old ones.

Make a npm install && node app.js and put a POST request to localhost:4000 . Cors is enabled so you could post from every origin.

<form action="http://localhost:4000/" method="POST" enctype="multipart/form-data">...</form>
